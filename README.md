# ToDo-App

Eine einfache ToDo-App mit HTML, CSS und JavaScript. Sie läuft direkt im Browser und speichert Aufgaben lokal im `localStorage`.

## Start

Öffne `index.html` direkt im Browser oder starte einen kleinen lokalen Server:

```bash
python3 -m http.server 8000
```

Danach ist die App unter `http://localhost:8000` erreichbar.

## Funktionen

- Aufgaben hinzufügen
- Aufgaben als erledigt markieren
- Aufgaben löschen
- Nach allen, offenen und erledigten Aufgaben filtern
- Erledigte Aufgaben gesammelt löschen
- Aufgaben dauerhaft im Browser speichern
